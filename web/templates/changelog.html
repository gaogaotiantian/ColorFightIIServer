Changelog

2019/7/22 - Pre-season Phase 2

Gameroom rule change
  • Gameroom now has a new attribute - rank
  • Only matches in rank room will be recorded for ladder scores
  • Only verified users are allowed in rank room now
  • Two 2-player room, a 4-player room and an 8-player room is available as rank rooms
  • Two bot rooms - Musikverein and Acropolis is available for single player
Leaderboard change
  • Leaderboard is reset for phase 2
  • Total game played will be displayed on leaderboard
Client change
  • We recommend you to update your API to the latest, but the old one still works
  • Some memory leak fix for running the bot continuously
  • destroy_gold and destroy_forcefield are provided in Building
  • get_gameroom_list is provided in Colorfight to screen and join gamerooms
  • ExampleAI has a small piece of script to run the bot forever
  • Some of the features are not applicable to Java client now. We will make it work for Java soon
