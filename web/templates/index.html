Welcome to ColorFightII

ColorFightII is an open source artificial intelligence platform, created by Tian Gao

In ColorFightII, you need to write a bot to expand, develop and conquer. You need to keep a good balance between war and economy, and try to collect more gold in the end of the game.

We switched server from heroku to digitalocean for better scaling.

Please download/pull the latest bot because the old one links to heroku instead of the domain.
