GET STARTED

Github Repo

If you are familiar with github, you can clone the example AI repo:

git clone https://github.com/gaogaotiantian/ColorfightIIBot.git the/path/you/want/

Download

You can also download the example AI from our website directly.

RUN YOUR BOT

Python3

You need python3.6+ and you need to install the requirement package by

pip3 install -r requirements.txt

Then you can run you bot by:

python3 example_ai.py

If you experience python ssl.SSLError: [SSL: CERTIFICATE_VERIFY_FAILED] error on MacOX:

It's because Apple forked their own version of ssl and python3.6 has this weird problem with wss/https.

You can switch to python3.7 or check this link for a solution

Java

To run your AI with the API we provide, modify example_ai.java and do:

./run_game.sh

If you are using an IDE, add colorfightII.jar as an external library in the IDE.

If you need to change our API, you need maven

Edit our Java source code in src and do ./build_jar.sh to create you own API jar package.

You can also make an AI without the jar file. Open our project in your IDE as a MAVEN project, then create a main class in src/main/java/colorfightII. You can run main in your IDE directly.

CHECK YOUR AI IN GAMEROOM

Go to gameroom page and you'll find a list of rooms that you can join.

You can create your own room with your rules and admin password by clicking the add button at the end of gameroom list.

By default, exampleAI joins public room. You can check the game status in it

You can click the restart button on the game stat panel to restart the game

VERIFY YOUR ACCOUNT

To get your ranking score and appear in the leaderboard, you need a verified account.

Log in with your Google/Facebook/Github account. The log in process is through OAuth2 so Colorfight does not have access to you password or store it in any format. The only thing Colorfight knows is your email address with the account and the fact you are logged in.

After you log in, you should be directed to your dashbord. If not, you can always click MENU-DASHBORD on the top right corner of the website.

There is a section "Game Authentication Info". Here you need to specify your "Game Username" and "Game Password" and use these in your AI to join the game.

We highly recommend you against using your regular password here!

Colorfight store your username/password set in plain text and display it on your dashboard for your convenience. Even though we would never leak or sell your information, it is still the best practice to keep yourself safe by using a unique password(which even does not to be that strong) here.

If you join the game with verified account, your username on the user list should appear in a different color.

You will be ranked and your ranking will be displayed in your dashboard and leaderboard.

FAQ

Q: Why the game stopped at 500/500?

A: Colorfight is a round-based game and each single game lasts for certain rounds. The game stops after specified number of rounds(default 500) and shows the final result. You can click restart button in game stat panel to restart the game. Or you can configure the game to restart immediately after finish.

Q: Why click restart button won't work?

A: If a room is created by other players, they may specify an admin password for it. You can only start/restart/configure the game if you have the admin password.

Q: Where should I input my admin password?

A: If you have the admin password for the gameroom, you need to open the configuration panel and input the admin password in "Admin Password" section. You need to do this to start/restart/configure the game. If the gameroom does not have admin password(like public), you should leave it blank to control the room.
